개인정보·쿠키 안내

나만의 메계부가 어떤 정보를 브라우저에 저장하고, 선택 기능을 사용할 때 어떤 정보가 외부 서비스로 전달될 수 있는지 안내합니다.

최종 수정: 2026년 7월 21일

1. 브라우저에 저장되는 정보

재획, 보스, 드랍 아이템, 손익, 지출, 캐릭터 설정과 화면 설정은 기본적으로 이용 중인 브라우저의 로컬 저장소에 보관됩니다. 운영자는 이 로컬 기록을 직접 열람할 수 없습니다. 브라우저의 사이트 데이터를 삭제하면 로컬 기록도 함께 삭제될 수 있으므로 JSON 백업을 권장합니다.

2. NEXON Open API 이용 고지

본 서비스는 NEXON Open API를 이용하여 캐릭터 프로필, 경험치, 스케줄러 정보를 제공합니다. NEXON의 공식 서비스가 아닙니다. 이용자가 직접 등록한 API 키는 해당 브라우저에 저장되며 JSON 백업에는 포함하지 않습니다. 키가 노출된 경우 NEXON Open API에서 재발급해 주세요.

3. 암호화 클라우드 동기화

API 키 등록 후 클라우드 동기화를 사용하면 기록은 브라우저에서 AES-GCM 방식으로 암호화된 뒤 Cloudflare Workers 저장소로 전송됩니다. 서버에는 암호문과 동기화 식별자가 저장되며, 복호화는 같은 API 키를 가진 브라우저에서 수행합니다. 이 기능은 기기 간 기록 이동을 위한 선택 기능입니다.

4. 익명 커뮤니티 통계

통계 탭에서 이용자가 직접 참여를 켠 경우에만 이번 주 재획 생산 메소, 재획 기록 일수, 주차와 익명 식별자가 전송됩니다. 캐릭터명, API 키, 보스·드랍 상세와 전체 장부는 익명 통계에 전송하지 않습니다.

5. 방문 분석

사이트 품질과 이용 현황을 파악하기 위해 Cloudflare Web Analytics를 사용합니다. 이 과정에서 방문 시각, 페이지 조회, 브라우저·기기와 관련된 집계 정보가 처리될 수 있습니다. 자세한 내용은 Cloudflare 개인정보처리방침에서 확인할 수 있습니다.

6. 광고와 쿠키

광고 기능이 활성화된 경우 Google AdSense가 광고 제공, 빈도 관리, 부정 클릭 방지와 성과 측정을 위해 쿠키 또는 기기 식별자를 사용할 수 있습니다. 지역과 동의 상태에 따라 개인 맞춤 광고 또는 개인 맞춤이 아닌 광고가 표시될 수 있습니다.

  • Google 광고 설정 관리

7. 이용자가 관리할 수 있는 항목

  • 브라우저 설정에서 이 사이트의 로컬 저장 데이터와 쿠키를 삭제할 수 있습니다.
  • 통계 탭에서 익명 커뮤니티 통계 참여를 언제든 끌 수 있습니다.
  • 캐릭터 등록 탭에서 저장된 넥슨 API 키를 삭제할 수 있습니다.
  • 광고·쿠키 동의 메시지가 표시되는 지역에서는 해당 화면에서 선택을 변경할 수 있습니다.

8. 문의

개인정보 또는 사이트 동작과 관련된 문의는 GitHub 이슈로 남길 수 있습니다.

광고 승인 전이나 광고를 불러오지 못한 경우에는 광고 영역을 표시하지 않습니다. Android 앱과 로컬 실행 환경에서도 광고를 불러오지 않습니다.
